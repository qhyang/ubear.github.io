研究登陆系统

我估计做登陆系统的人，要么是本科生学弟，要么是他们有人故意的，给自己留下方便。哈哈，总之，不管怎么样，对我们来说简直好极了。

研究校园网的登陆系统的目的有这么几个：

  • 提交数据的地址
  • 提交数据的格式
  • 密码是明文传输还是加密传输，如果是加密，又是如何加密
