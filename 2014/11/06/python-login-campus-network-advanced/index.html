总结语法

平时用的Python不多。导致基本语法都快忘光了。在写这个代码片段时，遇到一些语法问题，现在总结如下：

  • 在函数里面要使用全局变量，需要用global来声明
  • python 里面没有switch语法，用if/elif/else替代，要注意不是else if或者elseif
  • python 中的列表，添加元素时，用的是append()函数，不是add()，这点要和Java的List等集合区别开来
  • python中threading.Thread类在初始化时，args参数是一个元组，注意我上面的代码写的是args=(i,)，注意后面加了个逗号。不然会被编译器认为是只传了一个参数而不是一个元组。

线程总结

这个里面用的是python中线程的函数方式。还有一种是继承thread.Thread类，重写run方法。不过对python的类不是很熟悉，就采用了函数的方法。采用函数方式，过程是：

  • 设置一个运行函数
  • 将运行函数作为参数传递到thread.Thread里面

需要注意的问题是，采用锁对共享变量进行限制。这里我限制了usefullist变量的访问，只允许一个线程执行完添加操作之后，其他想添加的线程才可以获得锁。

一个线程的join方法使得该线程结束之后才会调用另一个线程的join方法，这个过程中，主调线程被阻塞。这是在没有参数的情况下，如果有参数，则会等待一个线程参数的时间之后再调用下一个线程的join方法。
